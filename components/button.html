# Button

使用方法

type：String 定义按钮类型 有primary ， success ， info ， warning ， danger

circle：布尔值 定义圆角 默认false

disabled：布尔值 定义是否可点击 默认false

icon： String 定义图标 有delete，share，edit，loading

loading：布尔值 定义加载动画 默认值 false
